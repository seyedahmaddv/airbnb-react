## Setup

1. Clone the code:

    ```bash
    git clone https://github.com/seyedahmaddv/simple-airbnb.git
    ```

2. Navigate to the project folder:

    ```bash
    cd simple-airbnb
    ```

3. Install dependencies:

    ```bash
    npm install
    ```

4. Run the project:

    ```bash
    npm start
    ```
